# standalone
Running abap2UI5 in a single HTTP Handler.

#### Start

App: `z2ui5_cl_my_standalone_app`

API: `/sap/bc/z2ui5standalone?sap-client=001&app_start=z2ui5_cl_my_standalone_app`

#### Content
* ICF Node
* Handler Class
* Custom App (z2ui5_cl_my_standalone_app)
